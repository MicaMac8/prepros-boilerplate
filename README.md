# prepros-boilerplate
Little boilerplate for fast frontend projects using prepros for compiling


# Settings :

template engine :
  Jade (Pug)

CSS
preprocessor :
  Sass
PostCSS :
  Autoprefixer
reset :
  normalize.scss

Compiler :
  Prepros

Libraries :
  jQuery
  Event Emitter *

Style guides :
  JavaScript : Standard Happiness (with ";") (https://github.com/JedWatson/happiness)
  Sass : SMACSS

Linters :
  js-standard (Happiness)
  sass-lint
